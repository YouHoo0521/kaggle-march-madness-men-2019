This is the project site for my machine learning project in CS760 at UW-Madison. My group will take on the Kaggle competition for NCAA March Madness.

The goal of this competition is to predict winning probabilities for all matchups in this year's NCAA March Madness tournament. In addition to this supervised learning task, there is a side competition for best exploratory analysis of a competitiveness score.
